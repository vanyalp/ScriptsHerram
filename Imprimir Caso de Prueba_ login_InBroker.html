Resumen:

Este caso de prueba indica la forma correcta de acceso al sistema de InBroker.

 

 

Precondiciones:
Se requiere lo siguiente:
 
  1. Un navegador compatible (Chrome).
  2. Conectar con la url: http://200.77.226.6/InbrokerV5Web/Home/index.aspx
  3. Estar dado de alta en el sistema
  4. Contar con un usuario activo
  5. Contar con la contraseña correcta del usuario activo
  6. Contar con permiso de perfil
Nº:Pasos:Resultados Esperados:
1

Ingresar al navegador web  la siguiente URL:

http://200.77.226.6/InbrokerV5Web/Home/index.aspx

Mostrará la pantalla de "¡Bienvenido!" para iniciar de sesión.

En la parte superior derecha estarán los campos de:

  • Usuario (identificado con un icono de usuario)
  • Contraseña (identificado con un icono de una llave)

Y el botón con el ícono de un diplosoma, para acceder al sistema.

Tambien  un menú inhabilitado en la parte central con las opciones:

  • Poliza
  • Facturación
  • Tesoreria 
  • Siniestros
  • Honorarios 
  • Tablas de seguros
2

Ingresar el nombre de usuario.

Aparecerá el nombre de usuario capturado.

3

Ingresar la contraseña del usuario.

Aparecerá la contraseña del usuario capturado en puntos blancos con la misma cantidad de carácteres ingresados.

4

Dar clic en el botón con el ícono de diplosoma.

Se visualizará la pantalla principal del sistema ("home" , con el icono de una casa), la cuál contendrá en la parte superior derecha el siguiente dato:

  • Nombre de usuario (el cual tendrá que coincidir con el nombre de usuario ingresado al inicio de sesión).

así como los siguientes botones:

  • Botón de Manuales de Aplicación.
  • Botón de Cambiar Contraseña.
  • Botón de "Cerrar sesión".

y el menú habilitado en la parte central para acceder a cada sección con las opciones:

  • Póliza
  • Facturación
  • Tesoreria 
  • Siniestros
  • Honorarios 
  • Tablas de seguros
